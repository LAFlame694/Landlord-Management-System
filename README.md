# Landlord Management System

A Django-based system for managing apartments, units, tenants, rent, and roles (Landlord & Caretaker).

## Features
- Apartment & unit management
- Tenant assignment & vacancy tracking
- Role-based access (Landlord / Caretaker)
- Rent records & payments (in progress)

## Tech Stack
- Python
- Django
- SQLite (dev)
- HTML/CSS (Django templates)

## Setup
```bash
git clone https://github.com/YOUR_USERNAME/landlord-management-system.git
cd landlord-management-system
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
